# Digit Classifier

Digit Classifier is a C++/Qt desktop application focused on handwritten digit recognition using the MNIST dataset. The project combines a graphical user interface for dataset visualization with a custom neural network core implemented in modern C++.

The application supports loading MNIST image and label files, browsing samples through a thumbnail gallery, displaying selected digits in a custom viewer, and preparing a modular neural network architecture with configurable dense layers. The project is structured using a layered design, separating the user interface, dataset handling, image conversion, asynchronous loading, and machine learning core.

The goal of the project is to build a complete educational machine learning application from scratch, without relying on high-level ML frameworks, while keeping the codebase readable, modular, and extensible.
